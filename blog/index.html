Technical Handbooks

  • Understanding LLVM Transformation Passes (in Chinese)
    This is a Chinese book I wrote during my spare time as a Ph.D. student. The motivation behind this work stems from my belief that LLVM transformation passes are incredibly powerful tools, yet the existing documentation is often vague and lacks sufficient detail. In particular, this book aims to address that gap by providing: (1) An introduction to LLVM basics. (2) A quick overview of several fundamental compiler optimization techniques. (3) A detailed explanation of each LLVM (Version 17.0.0) transformation pass, including definitions, my interpretations, and illustrative code examples. I hope this book serves as a helpful resource for those looking to deepen their understanding of LLVM transformation passes. Enjoy reading!
